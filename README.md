### Prerequisites

In order to run this project you need:

<ul>
-A Git hub account
-Git bash
-Node JS
-Visual Studio Code as your code editor
</ul>

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone[ (https://github.com/AbdelazizSoliman/Todo-List.git)]
```
